# Bestfit 智能健康管理多端应用
基于微信开发者工具开发的AI驱动个性化健康管理应用，支持微信小程序、安卓APK、iOS IPA多端部署，可直接下载安装到手机使用。

##  核心特性
 **多端适配**：基于微信开发者工具统一开发，支持打包生成微信小程序、安卓APK、iOS IPA，多端数据同步互通
 **全场景记录**：饮食、运动、喝水、排便一站式健康数据记录
 **个性化方案**：基于用户身体指标与健身目标，自动计算每日热量目标，生成专属饮食运动方案
 **AI智能分析**：接入大模型，自动生成周/月健康总结与可落地优化建议，支持自然语言健康咨询
 **女性专属**：经期记录、排卵预测与专属健康建议模块

##  技术实现
- 开发框架：微信原生小程序 + 微信开发者工具多端适配
- AI能力：大模型API接入 + Prompt工程专项优化
- 开发模式：AI辅助编码（Cursor / Claude Code）全流程提效

## 部署方式
1.  **微信小程序**：微信开发者工具上传代码，提交微信公众平台审核发布
2.  **安卓APK**：微信开发者工具使用「生成安卓APK」功能打包，可直接安装到安卓手机
3.  **iOS IPA**：微信开发者工具使用「生成iOS IPA」功能打包，支持苹果手机安装
